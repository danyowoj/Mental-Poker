## Как играть?
Для игры понадобиться 3+ терминала.

1. В первом терминале запускаем сервер командой `python server.py`
2. В отдельных терминалах запускаем клиентов командой `python client.py`
3. В одном из клиентов создаем игру командой `create`
4. С остальных клиентов подключаемся к игре по ее ID командой `join game_1`
5. Все игроки объявляют готовность командой `ready` и игра начинается
